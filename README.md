# BoothCloud

Landing page for BoothCloud — DIY photo booth hire, delivered UK-wide.

**Live site:** _(deployed on Vercel)_

## Structure

- `index.html` — the landing page
- `project/assets/` — images used on the page

## Local preview

Any static server works. For example:

```bash
python3 -m http.server 8000
```

Then open http://localhost:8000.

## Deployment

Auto-deployed to Vercel on every push to `main`.
